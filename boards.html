Advisory Boards

We have two advisory boards: the Technical Advisory Board and the Community Advisory Board

Technical Advisory Board

Our role is provide directions and support for code standardization and homogenization.
Our outcome include templates for Jupyter Notebooks and GitHub readme files, coding guidelines, and the package ORMIR-MIDS for data structure
